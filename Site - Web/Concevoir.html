SAE - 22 : Le bureau d'étude

Ce jour la nous avons pris le rôle d'un bureau d'étude durant la semain de SAE. Notre rôle etant de préparer l'integration d'AGV et d'AMR dans le but d'une automatisation du centre industriel. Nous avions à notre dispositon un fichier calque "Questionnaire AGV 2022" que nous devions remplir pour d'une part avoir toute les informations nécessaire à cette intégration (mesure de distance, angle de pente, nombre de rack, débit internet, type de sol, charge a soulever, nombre de station, rayonnages) mais aussi de savoir si cette integration etait possible celon les informations récupèrere sur site.

Aucune photo d'illustration ne sera présenté pour cause de confidentialité.
