## 🌍 Contexte

Projet réalisé lors du **Hackathon 2026 de l’IUT de Béziers**  
Thématique : **Transition écologique & développement durable**

---

## 💡 Le problème

Les informations sur les transports sont souvent :

- dispersées  
- peu fiables en temps réel  
- pas adaptées aux personnes à mobilité réduite  

---

## 🧠 La solution

**Smart Mobility Access** est une application web qui permet de :

- 🚆 Voir les transports disponibles en temps réel  
- 📍 Localiser bus, trains, taxis…  
- ♿ Identifier les transports accessibles PMR  
- ⏱️ Anticiper les déplacements  

---

## 🎯 Objectifs

- Améliorer l’accessibilité  
- Réduire le stress des déplacements  
- Encourager les transports écologiques  
- Centraliser l’information  

---

## 🧩 Fonctionnalités

- Carte interactive 🗺️  
- Filtres (type de transport, accessibilité)  
- Données en temps réel 📡  
- Notifications 🔔  

---

## 🏗️ Installation

```bash
git clone https://github.com/votre-repo/smart-mobility-access.git
cd smart-mobility-access
npm install
npm run dev
```
